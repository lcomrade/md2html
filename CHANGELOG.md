# Changelog
Semantic versioning is used (https://semver.org/).

## v1.0.0
First stable release.

The following Markdown elements are supported:
- Paragraph
- Header
- Text style
- Link
- Image
- Code quote
- Unordered list
- Numbered list
- Character shielding(`\*`, `\_`, `\~`, `\.`)
- Embedded HTML tags

And there is quite detailed documentation and examples.
